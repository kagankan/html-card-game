# 人間型キャラクター（探検家）仕様

## 基本コンセプト

小さくて親しみやすい、**丸みのあるデフォルメ探検家キャラクター**。
シンプルで落ち着いた見た目にし、装飾は少なめにする。

## 外見仕様

### 頭身・体型

- 約 **2.5〜3頭身**
- 全体的に **ころんとした丸いシルエット**
- 手足は短めで、やわらかい印象
- 細身すぎず、リアルすぎない
- 幼すぎず、でも親しみやすいデフォルメ感

### 顔

- 顔は **やわらかいクリーム色**
- 目は **縦長の黒い楕円が2つ**
- **鼻は描かない**
- **眉は描かない**
- **まつ毛は描かない**
- **チークは描かない**
- **口は基本なし**
  もしくは必要な場合のみ、ごく小さいニュートラルな口
- 表情は大きく誇張しない

### 帽子

- **青い探検家帽子**
- 形は **丸いドーム型**
- **短くやわらかいツバ**
- **帽子のてっぺんに小さな丸いボタン**
- 帽子の形は毎回固定する

### 服

- **青い探検家風のワンピース / つなぎ風の服**
- シンプルで装飾は少なめ
- 服の色は落ち着いた青で統一

### ベルト

- **茶色のベルトを胴まわりに真横一直線で入れる**
- **斜め掛けにしない**
- **ベルトの色は茶色一色**
- 不自然なグラデーションは付けない
- 正面ではベルト中央に丸い留め具があるデザイン

### リュック

- **小さめの茶色いリュック**
- シンプルで丸みのある形
- 服より少し落ち着いたブラウン系
- 装飾は少なめ

### 手足・靴

- 手は **ミトンのような単純な形**
- 足は短く、シンプル
- 靴は服と一体感のあるシンプルな形
- 指の描き分けはしない

## 色のルール

- 主な色は **青・茶・クリーム色**
- 全体は **やさしく落ち着いた低〜中彩度**
- 帽子と服は青系で統一
- ベルトとリュックは茶色系で統一
- 派手な多色使いはしない

## 雰囲気

- 親しみやすい
- かわいい
- 落ち着いている
- やさしい
- 学び・探索・発見に合う
- ミニマルで読みやすい
